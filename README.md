# weather-app
A dynamic weather application that allows users to search for any city and view real-time weather data. The app integrates with a weather API to display temperature, weather conditions, and location-based information. Built using HTML, CSS, and JavaScript with asynchronous API handling and deployed for public access.
## 🚀 Features
- Search weather by city
- Real-time API data
- Dynamic UI updates

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
